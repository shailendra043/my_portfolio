
# 3D Developer Portfolio

This is a 3D portfolio website built with React, Three.js, and Tailwind CSS.

## Features
- **3D Hero Section**: Interactive 3D visuals.
- **Experience Timeline**: Vertical timeline of work experience.
- **Tech Skills**: Interactive floating 3D balls for skills.
- **Projects**: 3D tilt effects on project cards.
- **Contact Form**: Integrated with 3D Earth model.
- **Responsive Design**: Works on mobile and desktop.

## Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Customization
-   Update `src/constants/index.js` with your own data.
-   Replace placeholder images in `src/assets` or update the URLs in `src/assets/index.js`.
